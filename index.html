Application settings

  • Create custom profiles with: work, break and long break intervals
  • Automatically start break\work intervals
  • Count time backwards
  • Custom work end and break end notification sounds
  • Minimize to system tray
  • Unclickable application while timer is running (you can always control application from system tray or taskbar)
  • and much more ...

Theme setting

  • Add custom fonts
  • Change application size
  • Select clock and shadow colors
  • Control clock opacity

Native application

With only 10mb download size and 40mb memory usage, YAPA 2 is the smallest pomodoro application out there.

FAQ

Navigate to My Documents > YAPA2 and create a new folder named Fonts. Put your .ttf fonts inside and restart the application.
